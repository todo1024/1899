课程介绍：

很多职场人一门心思放到工作上，却忽视了和同事、上下级的有效沟通，结果要么到处得罪人，要么不分场合的让人尴尬，要么把工作弄得一团糟。能力再强的人，也没法在职场上单打独斗，只有学会高效表达自己，才能更好的在职场脱颖而出！本课程将通过认知、理论、技巧及案例的详细阐述，围绕职场中最普遍的困境，教会你沟通，从此具备应有的职业素养！
